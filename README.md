DogCare Sistem Pakar Diagnosis Penyakit Anjing
DogCare adalah aplikasi Sistem Pakar berbasis aturan (rule-based) yang digunakan untuk membantu pemilik anjing dalam melakukan diagnosis awal penyakit anjing berdasarkan gejala yang dialami. Sistem ini dibuat sebagai tugas individu mata kuliah Sistem Pakar.

Tujuan
- Memberikan diagnosis awal penyakit anjing
- Membantu pemilik anjing mengambil keputusan awal
- Menerapkan konsep sistem pakar berbasis aturan

Metode
- Jenis sistem: Rule-Based Expert System
- Metode inferensi: Forward Chaining
- Bahasa pemrograman: Python
- Library: Experta

Penyakit yang Dideteksi
- Rabies  
- Canine Distemper  
- Skabies  

Fitur Sistem
- Input data pemilik dan anjing
- Input gejala (ya/tidak)
- Proses inferensi otomatis
- Output diagnosis dan saran
